# Лабораторная работа по дисциплине "Аналитик Данных"

Предметная область: погодная станция

Поля:
- температура (temperature)
- влажность (humidity)
- давление (presssure)
- скорость ветра (wind_speed)

## Что было проделано в рамках лабораторной работы
- Написан генератор данных на Python
- Написан Dockerfile для этого генератора для последующего использования в docker-compose
- Написан docker-compose для создания контейнеров:
    - Redash
    - PostgreSQL
    - Самого генератора
- Написан Jupyter Notebook в котором создается анализ данных из БД, а также два графика по данным из БД. 

## Запуск проекта
Для начала надо прописать поля для .env из .env.example, после выполнить комманду

```
docker compose up -d
```

поднимает все нужные контейнеры нужные для проекта. После запуска всех контейнеров в Redash необходимо добавить нужные графики/таблицы.
